# KOL Directory

A web scraping system for collecting KOL (Key Opinion Leader) data from social media platforms.

## 🚀 Quick Start

### Prerequisites
- Node.js 18+
- Git

### Installation
\`\`\`bash
# Clone repository
git clone https://github.com/YOUR_USERNAME/kol-directory.git
cd kol-directory

# Install dependencies
npm install

# Setup environment
cp .env.example .env
# Edit .env with your settings

# Run
npm start
\`\`\`
